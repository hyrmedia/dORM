Welcome to Dorm 0.0.2 !

Dorm is a PHP ORM (Object Relational Mapper).

www.getdorm.com

IMPORTANT NOTICE
----------------

This version is not stable and not recommended for use in a production
environment.

FEATURES
--------

* NO code generation
* NO need to modify classes
* NO need to write new classes
* NO need to modify your database structure
* NO naming conventions or file structure required
* ONE small configuration file needed

SYSTEM REQUIREMENTS
-------------------

* PHP 5.x
* MySQL 5.x
* APC (highly recommended)

INSTALL
-------

See http://www.getdorm.com/#getting-started or try the demo in demo/simple/
